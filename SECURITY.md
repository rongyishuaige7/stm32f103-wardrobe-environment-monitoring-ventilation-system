# 安全说明

## 适用范围

这是低压桌面教学原型，不是空气质量仪、家电控制器、消防/医疗设备或安全系统。禁止连接市电、安全关键负载或无人值守生产环境。

## 物理与测量安全

- PB0/PB1 不得直接驱动风扇、电机、继电器或其他大电流负载；
- MQ135 模块的 5 V 侧与模拟输出必须在接入 3.3 V MCU 前按精确模块核对；
- 传感器读数和原始阈值不能作为健康、安全或法规判断；
- 调整 opt-in 前先建立硬件急停/断电能力、确认驱动默认状态和回滚固件；
- 当前没有看门狗、冗余传感器、失效安全、故障锁存或执行器反馈。

## 数据与网络

当前固件没有网络、存储、云端、账号或远程控制功能，只通过本地 USART1 输出运行文本。串口日志仍可能暴露环境变化，不应在不可信终端长期采集。

## 报告问题

如启用了 GitHub 私密漏洞报告，请优先使用该入口。否则只提交经脱敏的最小问题描述，不要附带真实地址、账号、设备标识、本机路径、私有日志或凭据。

当前只维护 `main` 最新提交。
