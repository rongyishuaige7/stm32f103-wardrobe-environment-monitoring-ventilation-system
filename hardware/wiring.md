# 接线边界说明

> 本页是源码接口索引，不是已验证接线教程。具体 VCC、GND、上拉、分压、驱动和保护必须按手中精确模块重新设计和复核。

## 逻辑信号

| 信号 | STM32 引脚 | 方向 / 默认 | 边界 |
| :-- | :-- | :-- | :-- |
| DHT11 DATA | PB12 | 双向单总线 | 需要正确逻辑电压，可能需要外部上拉 |
| MQ AO | PA0 | ADC 输入 | 在接 MCU 前确认全范围不超过 ADC 允许电压 |
| OLED SCL | PB6 | 软件开漏 I²C | 确认模块上拉和供电 |
| OLED SDA | PB7 | 软件开漏 I²C | 同上 |
| UART TX | PA9 | MCU 输出 | 接 USB-TTL RX，3.3 V 逻辑并共地 |
| UART RX | PA10 | 当前仅初始化输入 | 固件不读取命令；不要注入 5 V |
| FAN IA/IB | PB0/PB1 | 默认保持未配置 | 只能接经确认的驱动器逻辑输入，不能接电机 |
| SWDIO/SWCLK | PA13/PA14 | 调试 | 使用匹配电平的 ST-Link |

## 供电域

```text
低压逻辑电源 ── STM32 / DHT11 / OLED（按精确模块额定值）
                         │
                         └── 逻辑信号与公共参考地

独立风扇电源 ── 经确认的电机驱动器 ── 低压风扇
                    ↑
              PB0/PB1（仅 opt-in）

MQ135 模块电源与 AO ── 先核对模块原理图和 AO 最大值 ── 必要分压/缓冲 ── PA0
```

该结构图不包含元件值、保护器件或连接授权。断电接线；有疑问时保持风扇断开并只使用 `safe-default`。
